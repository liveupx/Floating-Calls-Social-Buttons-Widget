A minimal setup with WhatsApp, Phone, and Email buttons. Click the floating button in the bottom-right corner!

How It Works

This example demonstrates the simplest possible widget configuration. We've added three contact buttons: WhatsApp, Phone, and Email.

The Code

<script>
  window.floatingWidgetConfig = {
    buttons: [
      { type: 'whatsapp', value: '1234567890', label: 'WhatsApp Us' },
      { type: 'phone', value: '+1234567890', label: 'Call Us' },
      { type: 'email', value: 'hello@example.com', label: 'Email Us' }
    ]
  };
</script>
<script src="../dist/widget.min.js" defer></script>
